# Kagōunga

Kagōunga Popeda Instant — makanan tradisional Maluku Utara yang praktis dan lezat.

## Project Setup

To run this project locally:

1. Clone the repository
2. Install dependencies:
    ```bash
    npm install
    ```
3. Start the development server:
    ```bash
    npm run dev
    ```

## Technologies

-   React
-   TypeScript
-   Vite
-   Tailwind CSS
-   shadcn/ui
